# Thermal Units

---

Display multiple temperature units at once. Largely identical to CelsiusAndFahrenheit, but updated for 1.1.

* Celsius: `Outdoors 10C [50F]`
* Fahrenheit: `Outdoors 50F [10C]`
* Kelvin: `Outdoors 283K [10C/50F]`

---

##### STEAM INSTALLATION
- **[Go to the Steam Workshop page](https://steamcommunity.com/sharedfiles/filedetails/?id=2006814687) and subscribe to the mod.**

##### NON-STEAM INSTALLATION
- **[Download the latest release](https://github.com/thecodewarrior/ThermalUnits/releases/latest) and unzip it into your *RimWorld/Mods* folder.**
